Bootcamp 2020: Relaxing and satisfying 2D animation with sound

This is my first programming project which has been coded using some of the tools and techniques that I learnt in the 2 week Programming Bootcamp at City (2020). This simple Processing project demonstrates a great understanding of the use of sound, colours and movement across the canvas using if statements. I created 2 bouncing shapes (one line and two balls) to give life to the project by using the X and Y axis of the canvas as reference points to make them move. In addition, if statements were used so that whenever one of the objects touches the width or height of the canvas, the objects travels in the opposite direction. To ensure that the shapes left trails behind whenever they move around the screen, I set a background() function on setup so that the trails stand out on a black background and also made sure that it wasn't added on draw so that the background doesn't keep updating every time the shapes move. Over time, the project should develop multiple rectangles of different colours and size all stacked on top of each other as shown in the pictures.

Further enhancements to the project

I enhanced the project by changing couple of values in the setup. First of all, I added another set of values for X and Y called "X2" and "Y2". Those values hold completely different numbers from the X and Y values that I originally set up for this project. This is because I want the shapes to move in a twisting and slower motion, so by having two different X and Y values I'll be able to obtain a much nicer looking effect. Furthermore, the last change that I made to the code to achieve the effect shown in the picture, was adding a scale() function on top of a smooth scaling size called sin().
